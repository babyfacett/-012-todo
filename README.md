# 12 ToDo アプリ

このアプリは、やるべきタスクを簡単に追加・管理できる ToDo リストです。タスクは Streamlit の `session_state` で管理しているため、ページを更新しても内容が失われません。

## 機能

- 新しいタスクを入力し、リストに追加
- タスクごとにチェックボックスで完了状態を切り替え
- 完了したタスクを一括削除

## 使い方

1. ターミナルから次のコマンドでアプリを起動します。

   ```bash
   streamlit run app.py
   ```

2. **「新しいタスクを入力してください」**の欄にタスクのタイトルを入力し、**「タスクを追加」**ボタンをクリックするとリストに追加されます。

3. タスクリストに表示された各タスクにはチェックボックスが付いており、完了したタスクはチェックを入れてください。

4. 完了したタスクをまとめて削除したい場合は、**「完了済みタスクを削除」**ボタンを押します。

タスク管理の基本を学ぶのに適した、シンプルな Streamlit アプリです。
