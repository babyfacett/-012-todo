# 12 ToDo アプリ

このアプリは、タスクを追加・完了チェック・完了済みタスクを削除できるシンプルな ToDo アプリです。セッション状態とコールバック関数を使ってタスクを管理し、「完了済みタスクを削除」を押すと即座に完了済みのタスクがリストから消えるように再実行を行っています【24130380429167†L46-L50】【269124077441344†L184-L188】。

## 機能

- テキスト入力欄にタスクを入力し、**Enter キー**を押すとタスクが追加されます。
- タスクごとのチェックボックスで完了状態のオン・オフを切り替えられます。
- **「完了済みタスクを削除」**ボタンを押すと、完了チェックされたタスクがすぐに削除されます。
- タスクが登録されていない場合は「現在登録されているタスクはありません。」と表示されます。

## 使い方

1. ターミナルで以下を実行してアプリを起動します。
   ```bash
   streamlit run app.py
   ```
2. 「新しいタスクを入力してください」にタスク内容を入力し、Enter を押すとタスクがリストに追加されます。
3. 完了したタスクはチェックボックスをオンにします。
4. 「完了済みタスクを削除」ボタンを押すと、チェックされたタスクが即座に削除され、リストが更新されます。

このアプリでは、タスクの削除後に `st.rerun()` を呼び出し、スクリプトを再実行することでUIを更新しています。`st.rerun()` はスクリプト実行を即座に停止し、再実行をキューに入れる関数で、画面を強制的に更新したい場合に有用です【269124077441344†L184-L188】。
